## Getting Started

1. Run `npm install`
2. Copy `.env.example` to `.env`
3. Update `VITE_API_BASE_URL` in `.env` to your backend URL
4. Set `VITE_ENABLE_MOCK_AUTH=false` when ready to use real backend auth endpoints
5. Run `npm run dev`

## API Environment Variables

- `VITE_API_BASE_URL`: Base URL for API requests
- `VITE_API_TIMEOUT_MS`: Axios timeout in milliseconds
- `VITE_ENABLE_MOCK_AUTH`: `true` for local demo auth, `false` for backend auth
